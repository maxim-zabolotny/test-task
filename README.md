# Test Task

### First make sure you have postgreSql installed and active

## Run the service
1. Open a new bash shell
2. ```cd backend```
3. ```npm install```
4. ```npm run start:dev```

## Run the app
1. Open a new bash shell
2. ```cd frontend```
3. ```npm install```
4. ```npm run start```


## Run the tests
1. Open a new bash shell
2. ```cd app```
3. ```npm run test_watch``` or ```yarn test_watch```
